1)git init
The git init command is used to initialize a new Git repository in a directory. 
It sets up the necessary Git configuration files and structures, making the directory a Git-tracked repository.
Basic ga local project ni remote ga push cheyali ante git init cheste .git folder create avutadhi appudu local folder setup git ki setup ayitadhi.

6)git push
a)Normal Push--->
-git push origin branch-name->Push changes to a remote branch

b)Push with Upstream--->
git push -u origin branch-name--->Set tracking for future push/pull

-u use cheste next direct ga git push or git pull use cheyachu.
-Pushes the branch and sets upstream tracking, so future pushes/pulls can be done with just git push or git pull.

d)git branch -a	List both local & remote branches
